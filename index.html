📚 OUI CSS Library — Beginner Documentation

This guide teaches you how to use every class in OUI library using simple examples. Copy the snippets, paste them in your HTML, and you’re good.

⚡ Fast — drop-in classes, no build step.
🧩 Composable — combine classes for layouts & components.
🎨 Themable — customize with CSS variables.

1) Install / Include

Add the CSS file to your page. You can inline it or link to it.


<link rel="stylesheet" href="https://omarxmario.github.io/OUI/ui.css" />

Tip: put it after your resets but before your app styles.

2) Theme Variables (Optional but recommended)

Your components read a few CSS custom properties. Override them once to match your brand.

:root{
  --trans-black: rgba(0,0,0,.18);
  --text-color: #0f172a;        /* text on components */
  --element-bg: #ffffff;        /* surface color for .def, .card */
  --border-col: #d0d5dd;        /* borders for .bor */
  --none: transparent;          /* used by .trans and .ico-btn */
}

3) Spacing Utilities

Padding utilities add space inside an element. Numbers are in pixels. Example: p-l-16 = padding-left:16px.

• Left Padding — p-l-[1..100]

<div class="p-l-16 demo-box">This box has 16px left padding</div>
<div class="p-l-40 demo-box">This box has 40px left padding</div>

• Right Padding — p-r-[1..100]

<div class="p-r-20 demo-box">This text moves away from the right edge by 20px</div>

• Bottom Padding — p-b-[1..100]

<button class="btn p-b-12 bor">Extra bottom padding (12px)</button>

Beginner tip: If you need top padding, wrap with a container and use p-b-* on a child, or add a top utility in your CSS similar to the others (pattern is the same).

4) Gap Utilities for Flex Layouts — g-[1..50]

gap adds space between flex items. Use it on any .r, .r-r, .c, or .c-r container.

<div class="r g-20">
  <div class="demo-box">Item A</div>
  <div class="demo-box">Item B</div>
  <div class="demo-box">Item C</div>
</div>

5) Flexbox Layout Helpers

Direction

  • .r — row (left → right)
  • .r-r — row-reverse (right → left)
  • .c — column (top → bottom)
  • .c-r — column-reverse (bottom → top)
<div class="r g-10">
  <div class="demo-box">1</div>
  <div class="demo-box">2</div>
  <div class="demo-box">3</div>
</div>

Horizontal distribution (justify-content)

  • .jus-st — start
  • .jus-ce — center
  • .just-ed — end
  • .jst-sb — space-between
  • .jst-sa — space-around
  • .jst-ev — space-evenly
<div class="r jst-sb">
  <div class="demo-box">Left</div>
  <div class="demo-box">Right</div>
</div>

Vertical alignment (align-items)

  • .ali-st — align start
  • .ali-ce — align center
  • .ali-ed — align end
<div class="r ali-ce" style="height:80px; background:#0b0f24; border:1px solid #1a2146; border-radius:12px;">
  <div class="demo-box">Centered vertically</div>
</div>

6) Components

• Card — .card (+ .def skin)

A simple content container with padding, rounded corners, and a soft shadow. Use .def to apply the default theme colors.

<article class="card def">
  <h4 style="margin:6px 0">Card title</h4>
  <p class="muted">Cards are great for grouping content.</p>
</article>

• Buttons — button, .btn, .ico-btn, .n-b, .bor, .trans

  • button base resets border/outline.
  • .btn adds padding + radius.
  • .ico-btn makes a minimal icon-style button.
  • .n-b removes borders; .bor adds a themed border.
  • .trans forces transparent background.
<button class="btn def bor">Primary</button>
<button class="btn trans bor">Ghost</button>
<button class="ico-btn bor" aria-label="Settings">⚙️</button>

• Container — .cont

Rounded top corners and scrollable content. Good for panels or drawers.

<section class="cont def" style="max-height:130px">
  <p>Scrollable content here…</p>
  <p>Add more paragraphs to see the overflow auto.</p>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
  <p>Vivamus luctus, massa sed fringilla blandit, dui elit mattis lectus…</p>
</section>

7) Compose Utilities Together

<div class="r g-16 ali-ce">
  <article class="card def p-l-20 p-r-20 p-b-16">
    <h4 style="margin:6px 0">Profile</h4>
    <p class="muted">Combine spacing + layout + components easily.</p>
    <div class="r g-10">
      <button class="btn def bor">Follow</button>
      <button class="btn trans bor">Message</button>
    </div>
  </article>

  <article class="card def p-l-16 p-r-16 p-b-12">
    <h4 style="margin:6px 0">Stats</h4>
    <div class="r jst-ev g-10">
      <span class="chip">Posts 120</span>
      <span class="chip">Likes 4.2k</span>
      <span class="chip">Friends 345</span>
    </div>
  </article>
</div>

8) Quick Reference (Cheat Sheet)

Spacing

p-l-[1..100], p-r-[1..100], p-b-[1..100]

All numbers are pixels.

Gap

g-[1..50] — use on flex containers.

Layout

.r, .r-r, .c, .c-r

.jus-st, .jus-ce, .just-ed, .jst-sb, .jst-sa, .jst-ev

.ali-st, .ali-ce, .ali-ed

Components

.card, .def, .btn, .ico-btn, .cont

Helpers: .bor, .n-b, .trans

9) Common Mistakes & Tips

  • Spacing doesn’t work? Check that you used the right side: p-l- for left, p-r- for right, p-b- for bottom.
  • Gap not working? Make sure the parent has a flex class like .r or .c.
  • Colors look off? Override the CSS variables in :root to match your theme.
  • Need top padding? Add a p-t-* utility following your pattern, or wrap children and use p-b-* strategically.
  • Icon-only buttons: use .ico-btn + .bor for a minimal affordance and set aria-label.
